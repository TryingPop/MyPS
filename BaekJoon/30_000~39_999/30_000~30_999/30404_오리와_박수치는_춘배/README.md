## 문제
춘배는 오리를 보러 왔다. 오리는 춘배가 있는 동안 총 N번 "꽥꽥" 소리를 낸다. 오리의 소리를 듣고 감동받은 춘배는 오리에게 박수를 쳐준다.<br/>
오리가 X_i초에 "꽥꽥" 소리를 낸다면 소리를 들은 춘배는 오리에게 X_i초 이상 X_i+K 초 이하에 한 번 이상 박수를 쳐야한다.<br/>
만약 오리가 소리를 낸 X_i초부터 X_i+K초 사이에 한 번도 박수를 쳐주지 않는다면 실망한 오리는 집으로 가버린다. 예를 들어 K=2, X_i = 5라면 5초, 6초, 7초 중 최소 한번은 박수를 쳐야 한다.<br/>
오리가 집으로 가지 않도록 춘배가 박수를 쳐줄 때 박수를 최소 몇 번 쳐야 하는지 구해보자.<br/>


## 입력
첫째 줄에 오리가 "꽥꽥" 소리를 내는 횟수 N와 정수 K가 공백으로 구분되어 주어진다.(1 ≤ N ≤ 100,000, 0 ≤ K ≤ 10^6)<br/>
둘째 줄에 오리가 "꽥꽥" 소리를 내는 시각 X_1, X_2, ..., X_N이 공백으로 구분되어 주어진다. X_i는 서로 다르며 오름차순으로 주어진다.<br/>


## 출력
오리가 집으로 가지 않도록 춘배가 박수를 쳐줄 때 박수를 최소 몇 번 쳐야 하는지 출력한다.<br/>


## 예제 입력
3 3<br/>
1 3 7<br/>


## 예제 출력
2<br/>


## 풀이
오리가 X초에 꽥꽥 소리를 낸다면 X + K초에 쳐야 X ~ X + K 사이에 있는 꽥꽥소리에도 반응하여 박수친것이 되기에 늦게 치는게 좋다.<br/>
X는 정렬되어 오므로 반응하지 않은 소리에 대해 최대한 늦게 박수를 치는 것이 좋다.<br/>
만약 Y초에 꽥꽥 우는게 X + K이하라면 X + K에 박수를 치기에 반응하는 것이므로 넘어가면 된다.<br/>
이렇게 진행하며 나온 박수치는 횟수가 정답이 된다.<br/>
