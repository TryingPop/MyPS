## 문제
동우는 길이 N의 수열 A=[ A_1, A_2, ..., A_N]을 분석하고 있다. 동우는 한 번의 시행으로 수열 A의 한 항 A_i를 원하는 정수로 바꿀 수 있다. 귀찮음이 많은 동우는 가능한 최소한의 시행으로 모든 이웃한 항의 차이를 M 이상으로 만들고 싶어 한다. 즉, 모든 1 ≤ i < N에 대하여 |A_(i+1) - A_i| ≥ M으로 만드는 것이 목표이다. 동우를 도와 필요한 최소 시행 횟수를 구해보자.<br/>


## 입력
첫 번째 줄에 정수 N(1 ≤ N ≤ 10^6)과 M(1 ≤ M ≤ 10^12)이 공백으로 구분되어 주어진다.<br/>
두 번째 줄에 N개의 정수 A_1, A_2, ..., A_N (1 ≤ A_i ≤ 10^12)이 공백으로 구분되어 주어진다.<br/>


## 출력
첫 번째 줄에 모든 이웃한 항의 차이를 M 이상으로 만들기 위해 필요한 최소 시행 횟수를 출력한다.<br/>
만약 불가능하다면 -1을 출력한다.<br/>


## 예제 입력
9 3<br/>
1 2 2 1 3 3 7 7 1<br/>


## 예제 출력
4<br/>


## 풀이
짝수항을 -M으로 모두 바꾸면 첫 입력은 양수만 주어지기에 이웃한 항의 차이는 모두 M 이상이 된다.<br/>
그래서 불가능한 경우는 존재하지 않는다.<br/>


이제 변환할 수 있는 최솟값을 찾아야 한다.<br/>
i번과 i + 1번의 인접한 항의 차이가 M 이하인 경우 i + 1번항을 -M으로 바꾼다.<br/>
그러면 i번째 항은 양수이므로 인접한 항의 차이가 M이상이 된다.<br/>
또한 i + 1과 i + 2역시 인접한 항의 차이가 M이상이 보장된다.<br/>


해당 방법으로 변형하는 것이 양쪽 인접한 항을 바꿀 수 있는 최소의 방법 중 하나이다.<br/>
이렇게 해당 방법으로 최소가되게 변형해가면 그리디로 최소가 됨이 보장된다.<br/>
코드에서는 주어지는 값이 10^12를 넘지 못하므로 -M이 아닌 10^15로 했다.<br/>
