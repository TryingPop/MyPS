## 문제
branchorama 나무는 특이한 규칙을 가지고 성장합니다. 어린 branchorama 초목은 하나의 잎을 꼭대기에 가진 가는 묘목이며, 그 잎에는 생장점이 있습니다. 성장하는 계절 동안 나무의 생장점들은 여러 개의 가지로 나뉘게 되며, 성장이 끝나면 각 가지는 생장점을 가진 하나의 잎을 꼭대기에 매달게 됩니다. 놀랍게도 같은 나무의 모든 생장점들은 같은 숫자(splitting factor)의 가지로 나뉘며, 그 숫자는 해가 지남에 따라 변합니다.<br/>
branchorama 나무는 과밀하게 성장하는 경향이 있습니다. 따라서 Brown 씨는 매 겨울마다 과도하게 성장한 나무들의 가지를 쳐냅니다.<br/>
branchorama 나뭇잎은 굉장히 크고 광합성에 유리하지만, 오직 생장점이 온전히 보존된 가지의 끝에만 달립니다. 따라서 나무가 버티지 못할 정도로 가지를 쳐내는 일은 없어야 합니다.<br/>
Brown 씨는 각 나무가 몇 개의 잎을 가졌는지 알고 싶어합니다. 나뭇잎을 일일이 세는 것은 지루하기 때문에, 각 해(level) 성장기의 splitting factor와 그 해 겨울에 쳐낸 가지의 수를 이용해 Brown 씨에게 나뭇잎의 수를 알려주세요.<br/>


## 입력
입력의 각 줄은 하나의 branchorama 나무를 의미합니다.<br/>
각 줄은 나무의 나이 a(1 ≤ a ≤ 20)로 시작하며, 그 뒤로 2a 개의 정수가 공백을 두고 주어집니다. 2a 개의 정수는 splitting factor와 가지치기 한 가지의 수가 level 별로 나열된 것입니다.<br/>
마지막 줄로  '0'이 주어지며 더 이상의 입력은 없습니다. '0'은 처리하지 않습니다.<br/>


## 출력
각 나무에 대하여 나무에 달려있는 잎의 수를 한 줄씩 출력하세요. 나뭇잎의 수가 signed 32-bit integer를 초과하지 않는다고 가정해도 좋습니다.<br/>


## 예제 입력
1 3 0<br/>
2 3 0 2 0<br/>
3 3 0 2 0 2 0<br/>
3 3 0 2 1 2 3<br/>
2 4 1 3 4<br/>
4 5 0 5 1 5 2 5 101<br/>
0<br/>

## 예제 출력
3<br/>
6<br/>
12<br/>
7<br/>
5<br/>
489<br/>


## 힌트
처음에 심는 것은 가지 하나에 잎이 하나 달린 묘목입니다.<br/>


## 풀이
맨 위에 있는 가지의 갯수를 찾아야 한다.<br/>
상황만 이해하면 정답 구하기는 쉽다.<br/>

4번째 예제 3 3 0 2 1 2 3를 보자.<br/>
맨 앞의 3은 level이 3이라는 의미다.<br/>
나무의 상태는 다음과 같다.<br/>

	??? (3단계)
	??? (2단계)
	??? (1단계)
	．｜．
	(시작 가지)


그리고 다음으로 오는 두 개의 숫자 3, 0은 뻗쳐나간 가지가 각각 3개이고,<br/>
이 중에 잘라낸 것은 0개라는 의미다.<br/>

	??? (3단계)
	??? (2단계)
	．＼｜／． (1단계)
	．．｜．．
	(시작 가지)

다음 2개의 숫자 2, 1은 뻗쳐나간 가지가 각각 2개이다.<br/>

	??? (3단계)
	＼｜＼．／｜／ (2단계)
	．＼．｜．／． (1단계)
	．．．｜．．．
	(시작 가지)

그리고 여기서 1개의 가지를 잘라낸다는 의미이다.<br/>

	??? (3단계)
	＼｜＼．／｜． (2단계)
	．＼．｜．／． (1단계)
	．．．｜．．．
	(시작 가지)

다음 2개의 숫자 2, 3은 뻗쳐나간 가지가 각각 2개이다.<br/>

	＼｜＼｜．＼｜．｜／．｜／ (3단계)
	．＼．｜．．＼．／．．｜． (2단계)
	．．．＼．．．｜．．．／． (1단계)
	．．．．．．．｜．．．．．
	(시작 가지)

이후 3은 여기서 3개를 잘라낸다는 의미이다.<br/>

	＼｜＼｜．＼｜．｜．．．． (3단계)
	．＼．｜．．＼．／．．｜． (2단계)
	．．．＼．．．｜．．．／． (1단계)
	．．．．．．．｜．．．．．
	(시작 가지)

그러면 나무의 모양은 위와 같다.<br/>
여기서 찾아야 하는건 3단계에 .이 아닌 선분의 갯수 7이다.<br/>


아래 가지에서 곱셈 먼저 하고 빼면서 나아가는 것이므로<br/>
어느 가지를 잘라도 모양만 다를 뿐 각 단계의 가지의 갯수는 같은 값을 가짐을 알 수 있다.<br/>
