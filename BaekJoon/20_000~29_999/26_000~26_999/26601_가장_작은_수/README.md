## 문제
#### 약수를 2^N개 갖는 양의 정수 중 가장 작은 수를 구하자.

## 입력
#### 첫째 줄에 정수 N이 주어진다. (0 ≤ N ≤ 111,222)

## 출력
#### 첫째 줄에 약수를 2^N개 갖는 양의 정수 중 가장 작은 수를 출력한다. 단, 수가 너무 커질 수 있으니 2,000,003으로 나눈 나머지를 출력한다.

## 예제 입력
7<br/>

## 예제 출력
83160<br>

## 힌트
  1. 약수는 어떤 수를 나누어떨어지게 하는 수를 말한다. 예를 들어 6의 약수는 1, 2, 3, 6이다.
  2. 2,000,003은 소수다.
