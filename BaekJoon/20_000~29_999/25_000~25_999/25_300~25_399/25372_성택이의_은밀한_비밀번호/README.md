# C#

## 난이도 : 브론즈 5

## 알고리즘 분류
  - 구현
  - 문자열

## 제한조건
  - 시간 제한 : 1초
  - 메모리 제한 : 1024 MB

## 문제
부산사이버대학교 학생 성택이는 엄마의 의뢰를 받아 주어진 문자열이 현관문 비밀번호에 사용 가능한지 알아내야 한다. 성택이는 공부해야 하므로 우리가 도와주자!<br/>
사용할 수 있는 비밀번호의 규칙은 다음과 같다.<br/>

  1. 비밀번호는 6자리 이상 9자리 이하여야 한다.

예를 들어, 123124는 올바른 비밀번호이고, 1202727161은 잘못된 비밀번호이다. 문자열이 주어졌을 때 현관문 비밀번호로 사용할 수 있는지 판단하자.<br/>


## 입력
첫째 줄에 문자열의 총개수 N이 주어진다.<br/>
둘째 줄부터 N개의 줄에 걸쳐 숫자, 영어 대소문자로만 구성된 문자열이 주어진다.<br/>


## 출력
줄마다 사용할 수 있는 비밀번호면 yes, 그렇지 않으면 no를 출력한다.<br/>


## 제한
  - 1 ≤ N ≤ 1,000 
  - 문자열의 길이는 1자리 이상 20자리 이하이다.


## 예제 입력
3<br/>
1245125<br/>
asdij<br/>
120318739721<br/>


## 예제 출력
yes<br/>
no<br/>
no<br/>


## 풀이
줄로 입력받고 문자열의 길이가 6 ~ 9자리인지 확인하면 된다.<br/>
