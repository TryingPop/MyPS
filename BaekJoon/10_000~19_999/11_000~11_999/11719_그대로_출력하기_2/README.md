# C#

## 난이도 : 브론즈 3

## 알고리즘 분류
  - 구현
  - 문자열

## 제한조건
  - 시간 제한 : 1초
  - 메모리 제한 : 256 MB

## 문제
입력 받은 대로 출력하는 프로그램을 작성하시오.<br/>

## 입력
입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. 각 줄은 100글자를 넘지 않으며, 빈 줄이 주어질 수도 있고, 각 줄의 앞 뒤에 공백이 있을 수도 있다.<br/>

## 출력
입력받은 그대로 출력한다.<br/>

## 예제 입력
    Hello<br/>
<br/>
Baekjoon     <br/>
   Online Judge    <br/>

## 예제 출력
    Hello<br/>
<br/>
Baekjoon     <br/>
   Online Judge    <br/>


## 풀이
문자열을 하나씩 입력받고 바로 출력했다.<br/>
